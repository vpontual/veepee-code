Write a TypeScript function `isPrime(n: number): boolean` in a file named `solution.ts`.

Requirements:
- Export it as a named export: `export function isPrime(...)` (or `export const isPrime = ...`)
- Returns `true` if `n` is prime, `false` otherwise
- Must handle edge cases: `n <= 1` → false, `n === 2` → true, negatives → false, non-integers → false

Use the `write_file` tool to create `solution.ts`. Do not write any other files. When the file is complete and correct, stop.
